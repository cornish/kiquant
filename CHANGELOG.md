# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-02-03

### Added
- Custom application icon for executable

## [0.1.0] - 2025-02-03

### Added
- Initial release
- Positive and negative marker annotation
- Rectangle and lasso selection modes
- Undo/redo with per-image history
- Quick mode for fast quantification without project setup
- WSI-style viewer with overview map
- Zoom toward cursor with mouse wheel
- Pan mode and middle-click panning
- Field guide (marker bounding box) with toggle
- Context menu with mode switching and marker operations
- CSV export with proliferation index calculation
- Project save/load with auto-save on navigation
- Save-before-close warning for unsaved changes
- Viewport state preservation between images
- Keyboard shortcuts for all major operations
- About dialog
