# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2025-02-03

### Added
- HTML About dialog centered in app window with icon
- Real-time proliferation index display in toolbar (1 decimal place)
- Vector overlay for markers, guide, and selection (crisp at any zoom)
- Yellow highlight for selected markers

### Fixed
- Icon marker colors (green for DAB positive, red for hematoxylin negative)
- Updated copyright and GitHub URLs

## [0.1.3] - 2025-02-03

### Added

## [0.1.0] - 2025-02-03

### Added
- Initial release
- Positive and negative marker annotation
- Rectangle and lasso selection modes
- Undo/redo with per-image history
- Quick mode for fast quantification without project setup
- WSI-style viewer with overview map
- Zoom toward cursor with mouse wheel
- Pan mode and middle-click panning
- Field guide (marker bounding box) with toggle
- Context menu with mode switching and marker operations
- CSV export with proliferation index calculation
- Project save/load with auto-save on navigation
- Save-before-close warning for unsaved changes
- Viewport state preservation between images
- Keyboard shortcuts for all major operations
- About dialog
